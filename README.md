 Personal Expense Tracker

 web app to help users manage their daily expenses.

 Features
- Add income and expense records
- View total balance
- Edit and delete records
- Simple and intuitive UI

Technology Stack
- HTML
- CSS
- JavaScript
- LocalStorage for data persistence

 Live Demo
https://kodingkole.github.io/personal-expense-tracker

 How to Use
1. Open index.html in your browser
2. Add your income and expenses
3. See your updated balance instantly
4. Edit or delete records if needed
